# 🧮 Calculator - C# Windows Forms

## 📌 Giới thiệu
Calculator là một ứng dụng **máy tính cơ bản** được xây dựng bằng **C# Windows Forms**.  
Ứng dụng hỗ trợ các phép tính số học thông dụng như cộng, trừ, nhân, chia với giao diện đơn giản, dễ sử dụng.

## ✨ Tính năng
- Cộng, trừ, nhân, chia các số thực.
- Xóa toàn bộ phép tính.
- Giao diện thân thiện với người dùng.


## 🛠 Công nghệ sử dụng
- **Ngôn ngữ:** C#
- **Nền tảng:** .NET Framework / .NET (Windows Forms)
- **IDE:** Visual Studio

## 📂 Cấu trúc thư mục


calculator/
├── Setup1/           # Thư mục cài đặt
├── calculator/       # Mã nguồn chính của ứng dụng
├── setup/            # Tệp hỗ trợ cài đặt
├── calculator.sln    # File Solution của Visual Studio



## 🚀 Cách chạy dự án
1. **Clone repo**:
   ```bash
   git clone https://github.com/USERNAME/calculator.git
````

2. **Mở file** `calculator.sln` bằng **Visual Studio**.
3. **Chạy ứng dụng** bằng cách nhấn **F5** hoặc nút **Start** trong Visual Studio.
